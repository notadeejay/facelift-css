You are a project manager for Brite Connections, a software company that creates applications intended to connect people of marginalized groups to support groups, health care, and mental health services. You are overseeing the creation of a new app that connects LGBTQ+ people with support groups and mental health services.

In this lab, you will create a resource breakdown structure for the project. The resource breakdown structure will provide a visual overview of the resources needed to complete the project.

The project is the development of a mobile app that connects LGBTQ+ people with support groups and mental health services.

In the example above, you started with a price per night of $149 and a "Private room". So your code should print:

2. According to the if statement, if the price per night is under $150, then your program will set the message to "This accommodation is affordable for Brooklyn" and continue on to evaluate nested if-statement. If the second if-statement is False, it will execute the statement below the else-statement. Setting message equal to "This accommodation is affordable, but it isn't the right type"

In the example below, the price per night is $149 and the room type is "Shared".

                                                            This accommodation is affordable, but it isn't the right type.

3. According to the if statement, if the price per night is not under $150, then your program will skip right to the else-statement, and set the message to "This accommodation is not within your budget" before printing it

In the example below, the price per night is $160 and the room type is "Private room".

                                                            This accommodation is outside of your budget.

You full solution should look similar to this:

                                                            if price_per_night < 150:
                                                                message = f"This accommodation is affordable for {neighborhood}."
                                                                if room_type != "Shared":
                                                                    message = f"This {room_type} is affordable for {neighborhood}."
                                                                else:
                                                                    message = "This accommodation is affordable, but it isn't the right type."
                                                            else:
                                                                message = "This accommodation is outside of your budget."
                                                            print(message)
